# Neovim Commands

## Default Commands

```shell

dw  = delete word
diw = delete inner word
dd  = delete line

v = select characters
V = select lines

```

`/<your-string>` = search recursive

## Macro

`q<char>` = start recording

`q` = stop recording

`@<char>` = play recording

NOTE: replace `<char>` with your desired choice of character/letter
